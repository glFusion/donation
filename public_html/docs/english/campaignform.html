Donation Plugin

Campaign Edit Form
Option Description
Campaign ID Enter the ID for this campaign. This must be unique and can be used in autotags to include donation buttons in articles.
Campaign Name Enter a name for this campaign. This will be shown as the campaign title in listings and blocks.
Starting Date Enter the starting date for this campaign, formatted as YYYY-MM-DD. You can click on the calendar icon to use a date picker.

Donations will not be solicited for this campaign before this date. Leave this field blank to start immediately.

Ending Date Enter the ending date for this campaign, formatted as YYYY-MM-DD. You can click on the calendar icon to use a date picker.

Donations will not be solicited for this campaign after this date. Leave this field blank to have this campaign run indefinitely.

Description Enter a description for this campaign.
Short Description Enter a short description for this campaign. This will be shown in the donation block and in the main campaign donation page, before the "Description" below.
Goal Enter the total goal for this campaign. This is used to create the bar graph in the campaign block. If there is no stated goal, leave this field blank and no bar graph will be created.
Stop after goal is reached? If this box is checked, donations will not be solicited for this campaign once the goal amount is reached. If unchecked, contributions will be selected indefinitely (or until the stated ending date).
Show progress bar in blocks? Check this box to have a bar included in the contribution block indicating the percentage of the goal that has been received. This will only be shown if the goal is greater than zero.

Note that if the PayPal plugin integration is not used, then the progress bar will not be updated unless the administrator manually enters the donations as they are received.

Enabled? Uncheck this box to disable the campaign. Disabled campaigns will not be shown in the public-facing lists or blocks.
