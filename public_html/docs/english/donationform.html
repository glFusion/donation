Donation Plugin

Donation Edit Form
Option Description
Date Enter the date of the donation. This must be formatted as YYYY-MM-DD. You can click on the calendar icon to use the date picker.
Campaign ID Select the campaign that to which this donation should be applied.
Contributor Select the site user who is making the contribution. For a non-member, select "Anonymous" and enter the full name in the space provided.
Amount Enter the amount of this donation.
Transaction ID Enter the transaction ID or receipt number for this donation.
Comment Enter any comment related to this donation.
