Donation Plugin

The Donation Plugin allows you to track donations to fundraising campaigns.

This is intended as a basic proof-of-concept showing how a plugin can provide items for sale via the PayPal plugin.

Donation Plugin Configuration Options

Main Settings
Option Description
Use Paypal Donation Button? Select "Yes" to use the "_donation" command with the Paypal button. Note that Paypal requires certain criteria to be met for donation buttions.

To use regular shopping-type buttons, select "No"
